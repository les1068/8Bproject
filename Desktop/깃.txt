1. git init 으로 .git  만들기
2. ls -al 로 확인
3. git add .
4. git commit -m "~~"  로 커밋
5. git branch -M main 로 main으로 변경
6. git remote add origin [주소] 로 내 레포지티로 연동
7. git push -u origin main으로 푸쉬
-------------------------------------------
git checkout -b [] 로 브랜치생성
git checkout [] 로 브렌치 이동
git remote rm origin 으로 연결 삭제
git pull origin master 로 코드 작성중에 가져올수있음
가져와서 수정 후 사용법 , GUI 사용법
